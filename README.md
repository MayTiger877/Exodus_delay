# Exodus Delay

![Build Status](https://github.com/MayTiger877/Exodus_delay/actions/workflows/build.yml/badge.svg)

A Delay effect plugin that gives each delay recpeat it's own charachter. Every delay line instance can have different mix level of modifiers & effects:

Gain, Pan, Distortion, Phaser and Reverb.

Each effect has its own parameters and designs, fully customizable;

All general delay settings are also customizable.
